您好，欢迎来到 FloatingBuds！

FloatingBuds 是一款轻量且简单的软件，可与您的好友进行基于 uri 的对话。

FloatingBuds 根据 GPLv3 许可证发布，按原样提供，我们对其误用不承担任何责任。

第一步，使用左侧面板密码和盐字段为配置文件中的每个朋友创建哈希。请记住手动设置盐值。

当您要使用有限的 Web 服务器或 phpfpm 用户在 PHP 进程上下文中运行 FloatingBuds 时，您必须遵循一些简单的指令以获得最佳的首次设置：
  1. 检查网络应用专用路径中“数据”文件夹的权限；并在配置文件中设置其路径。
  2. 完成配置文件的相应设置，具体为：
    • 适当配置上传的 APP_FILE_MAX_SIZE。
    • 根据需要配置 APP_BLOG_MAX_POSTS 属性。
    • 根据需要配置 PAGINATION 标志。
